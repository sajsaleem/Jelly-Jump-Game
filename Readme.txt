Jelly Jump game created on unity:
Used simple Cube available in Unity as Jelly because I was not able to find any asset,meshes to create jelly.
All assets used in game are created using Unity no external tool is used.

Game Obstacles:
two types of obstacles are created in game:
1. Sliding Obstacle
2. Rotating Obstacle 

Scripts:
1. GeneralSlideScript- Sliding Obstacle / Sliding Hazard
2. ShutScript (Child of GeneralSliderScript)- Rotating Obstacle/Shut Hazard
3. FollowScript - Attached with Camera to follow player in game
4. JellyScript- Jelly
5. GameController_Script - to control instantiation of Hazards and everything in game
6. WaterMove - script to move water that follows player in game.
